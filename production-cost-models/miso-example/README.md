# SIENNA Model of the MISO System

## Running the model

### Prerequisites
- You must have Julia 1.11 and SIENNA setup to run. See instructions here: https://nrel-sienna.github.io/PowerSystems.jl/stable/how_to/install/#install

### Important Files
- model-build.ipynb - builds the model from data files
- model-economic-dispatch.ipynb - runs the economic dispatch model
- model-resource-adequacy.ipynb - runs the resource adequacy (PRAS) model
